# 🧠 Meeting Intelligence NLP Model

This project uses NLP and Machine Learning techniques to classify meeting transcripts into categories such as:

- Decision
- Action
- Issue
- Deadline
- General

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NLP
- LinearSVC
- Matplotlib

## Features
- Text preprocessing
- Feature extraction
- Transcript classification
- Model evaluation
- Data visualization

## Learning Outcome
Gained hands-on experience in Natural Language Processing, Machine Learning workflows, and text classification techniques.
